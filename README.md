# Login-Screen
Login Page using HTML & CSS – Project Description
Project Overview
The Login Page is a modern, responsive, and visually appealing web interface developed using HTML5 and CSS3. The purpose of this project is to create a secure and user-friendly authentication page where users can enter their login credentials to access a website or application. The project demonstrates the use of HTML for structuring web content and CSS for designing an attractive and interactive user interface.
The design emphasizes simplicity, usability, and aesthetics, making it suitable for websites, web applications, and portfolio projects.
Technologies Used
HTML5 – Used to create the structure and layout of the login page.
CSS3 – Used to style the page, create animations, apply colors, and improve the overall appearance.
Google Fonts (Poppins) – Used to provide a clean and modern typography.
CSS Animations & Effects – Used to create glowing borders, hover effects, and smooth transitions.
